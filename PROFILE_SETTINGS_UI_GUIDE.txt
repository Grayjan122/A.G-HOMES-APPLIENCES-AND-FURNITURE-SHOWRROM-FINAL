Your new password must have:

✓ At least 8 characters
✓ At least 1 uppercase letter (A-Z)
✓ At least 1 number (0-9)
✓ At least 1 special character (@$!%*?&)

VALID EXAMPLES:
✅ MyPass123!
✅ Secure@456
✅ Test$Pass99

INVALID EXAMPLES:
❌ mypass123  (no uppercase, no special char)
❌ MyPass123  (no special character)
❌ MyPass!    (no number, too short)

🎯 USER SCENARIOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SCENARIO 1: User remembers old password
─────────────────────────────────────────────────────────────────────
1. Click "Change Password"
2. Enter old password
3. Enter new password
4. Confirm new password
5. Click "Save Changes"
6. Done! ✓

TIME: ~30 seconds


SCENARIO 2: User forgot old password
─────────────────────────────────────────────────────────────────────
1. Click "Change Password"
2. Click "Forgot Password?"
3. Verify email (pre-filled)
4. Click "Send Code"
5. Check email for code
6. Enter 6-digit code
7. Click "Verify Code"
8. Enter new password
9. Confirm password
10. Click "Reset Password"
11. Done! ✓

TIME: ~2-3 minutes (including checking email)
