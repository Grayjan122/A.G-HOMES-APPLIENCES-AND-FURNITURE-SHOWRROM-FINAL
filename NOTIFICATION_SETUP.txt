====================================
 NOTIFICATION SYSTEM SETUP GUIDE
====================================

STEP 1: CREATE DATABASE TABLE
------------------------------
1. Open phpMyAdmin: http://localhost/phpmyadmin
2. Select your database
3. Go to SQL tab
4. Run the SQL from file: database_notifications.sql

STEP 2: FILES CREATED
------------------------------
Frontend:
- app/Components/Notifications/NotificationBell.js
- app/Components/Notifications/notifications.css

Backend:
- C:\xampp\htdocs\capstone-api\api\notifications.php

Integration:
- Header component updated with NotificationBell

STEP 3: NOTIFICATION TYPES
------------------------------
1. Stock Request - Blue bell
2. Delivery - Green bell  
3. Out of Stock - Yellow bell
4. Payment Due - Cyan bell
5. Overdue - Red bell

STEP 4: FEATURES
------------------------------
- Real-time notifications (polls every 30 seconds)
- Unread count badge
- Mark as read by clicking
- Mark all as read button
- Role-based notifications
- Location-based notifications
- Beautiful dropdown UI
- Responsive design
- Auto-checks for low stock
- Auto-checks for payment due dates

STEP 5: HOW TO TEST
------------------------------
1. Run the database_notifications.sql script in phpMyAdmin
2. Refresh your application
3. You should see a bell icon in the header
4. Click the bell to open notifications

SETUP COMPLETE!
