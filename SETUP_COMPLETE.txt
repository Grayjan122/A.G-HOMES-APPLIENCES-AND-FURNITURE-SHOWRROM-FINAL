This will load your new Cloudinary credentials.


STEP 2: Test Image Upload
--------------------------
1. Open your browser: http://localhost:3000
2. Navigate to the Products page
3. Click "ADD PRODUCT+"
4. Fill in the product details:
   - Product Name
   - Category
   - Dimensions
   - Color
   - Price
   - Description
   - Material
5. Click "Choose File" and select an image
6. You should see an image preview
7. Click "Save"

------------------------------
✅ Product should appear in the list with the image

✅ Check browser console (F12) - no errors
✅ Image URL should start with: https://res.cloudinary.com


STEP 4: Check Cloudinary Dashboard
-----------------------------------
1. Go to: https://cloudinary.com/console
2. Click "Media Library" in the left menu
3. Navigate to folder: ag-homes/products
4. You should see your uploaded image!

If upload fails:

1. Check browser console (F12) for error messages
2. Make sure you restarted dev server after creating .env.local
3. Verify .env.local exists in project root
4. Check that image file is valid (jpg, png, webp, gif)
5. Try a smaller image (< 5MB)

Common errors:

"Cloudinary configuration missing"
→ Restart dev server: npm run dev

"Invalid credentials"
→ Check .env.local has correct API Secret
→ No extra spaces or quotes

"Upload failed"
→ Check browser console for specific error
→ Verify internet connection
→ Check Cloudinary account is active
