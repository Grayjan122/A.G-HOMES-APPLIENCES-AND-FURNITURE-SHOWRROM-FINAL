🕐 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. User logs in → Timer starts (1 hour)

2. User is active → Timer resets on each activity

3. After 59 minutes of inactivity → Warning modal appears

4. Warning modal shows:
   ┌─────────────────────────────────────┐
   │ ⚠️ Inactivity Warning               │
   │                                     │
   │ You have been inactive for too long!│
   │                                     │
   │ You will be logged out in:         │
   │           60s                       │
   │                                     │
   │ [✓ Stay Logged In] [✗ Logout Now]  │
   └─────────────────────────────────────┘

5. Options:
   a. Click "Stay Logged In" → Timer resets to 1 hour
   b. Click "Logout Now" → Immediate logout
   c. Do nothing → Auto logout after 60 seconds

6. On logout:
   - User status → "Offline"
   - Audit log created
   - Session cleared
   - Redirect to login page
