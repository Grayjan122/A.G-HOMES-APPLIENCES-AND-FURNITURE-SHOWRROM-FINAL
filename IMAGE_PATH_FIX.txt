The helper function checks the image path:

1. If path starts with "http://" or "https://"
   → It's a Cloudinary URL
   → Return as-is
   → Example: https://res.cloudinary.com/.../product_123.jpg

2. If path is a local path
   → It's from your old uploads folder
   → Return as-is (Next.js serves it from public folder)
   → Example: /uploads/products/image.jpg

3. If path is null or "Nothing as for now"
   → Return null
   → Shows "No Image" placeholder
