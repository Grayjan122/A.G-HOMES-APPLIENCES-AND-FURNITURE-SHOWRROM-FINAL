STATE MANAGEMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

React States:
  showWarning        boolean    Show/hide warning modal
  countdown          number     Countdown timer (60 → 0)

Refs:
  inactivityTimerRef    Timer ID for main 1-hour timeout
  warningTimerRef       Timer ID for 59-minute warning trigger
  countdownIntervalRef  Interval ID for countdown ticker
  lastActivityRef       Timestamp of last user activity

SessionStorage:
  tabHiddenTime      Timestamp when tab was hidden
  uncleanExit        Flag for browser crashes
  user_id            Current user ID
  baseURL            API base URL

COMPONENT LIFECYCLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Mount:
  ↓
  1. Add event listeners (mouse, keyboard, etc.)
  2. Add beforeunload listener
  3. Add visibilitychange listener
  4. Start inactivity timer
  5. Check for unclean exit flag
  ↓

Active:
  ↓
  1. Track user activity
  2. Reset timer on activity (throttled)
  3. Monitor tab visibility
  4. Wait for timeout or warning
  ↓

Warning:
  ↓
  1. Show warning modal
  2. Start countdown interval
  3. Wait for user action or timeout
  ↓

Cleanup/Unmount:
  ↓
  1. Remove event listeners
  2. Clear all timers
  3. Clear all intervals
  4. Set user offline (if browser closing)


ERROR HANDLING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Graceful Degradation:
  ┌─────────────────────────────────────────────┐
  │ If Backend API Fails:                       │
  │   → Log error to console                    │
  │   → Continue with logout anyway             │
  │   → User still redirected to login          │
  │   → No user-facing error shown              │
  └─────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────┐
  │ If sessionStorage Empty:                    │
  │   → Skip API calls                          │
  │   → Still clear and redirect                │
  │   → Prevent null/undefined errors           │
  └─────────────────────────────────────────────┘
