🚀 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Normal Login (Account Offline):
  Login → Backend checks → Status: Offline → ALLOW LOGIN ✅

Account Already In Use:
  Login → Backend checks → Status: Online → BLOCK LOGIN ⚠️
    ↓
  Show modal: "Account Already In Use"
    ↓
  Options:
    • Force Logout & Login (terminate other session)
    • Cancel (keep other session)
